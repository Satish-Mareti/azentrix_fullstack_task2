# Azentrix Full Stack Task 2 – Machine Learning Projects

This repository contains multiple end-to-end Machine Learning projects developed as part of the Azentrix Full Stack AI assignment. Each project demonstrates a complete ML workflow including data preprocessing, exploratory data analysis, model training, evaluation, and deployment-ready components.

---

## Projects Included

### 1. Iranian Churn Prediction
Predicts whether a customer is likely to churn using customer behavioral data.

**Highlights**
- Data preprocessing
- Feature engineering
- SMOTE for class balancing
- Model comparison
- Hyperparameter tuning
- Performance evaluation

**Algorithms Used**
- Logistic Regression
- Random Forest
- XGBoost

---

### 2. Loan Default Prediction
Predicts whether a loan applicant is likely to default.

**Highlights**
- Data cleaning
- Exploratory Data Analysis
- Feature Engineering
- Multiple model comparison
- Hyperparameter optimization
- Model evaluation

**Algorithms Used**
- Logistic Regression
- Random Forest
- Gradient Boosting

---

### 3. Telco Customer Churn Prediction
Predicts customer churn for a telecommunications company using customer subscription data.

**Highlights**

- Customer Risk Prediction

- Model Performance Visualization

**Algorithms Used**
- Logistic Regression
- Random Forest
- XGBoost

### 4. UCI Heart Disease Prediction
